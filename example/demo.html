Custom Scroll Bar

The element below is converted into a scrollable area by calling a jQuery function named 'customScrollBar' on it. This function returns an object with three functions:

  1. getScrollbarWidth: Returns the scrollbarwidth for the current browser.
  2. update: Updates the sizing of the scrollbars. Use this if the node has been changed with the scrollbar active.
  3. cleanNode: Clean the scrollbar from the node, returning it to it's original state.

If your browser/OS is configured to not show scrollbars, the new scrollbars will NOT be applied

	window.csb = $elem.customScrollBar();
